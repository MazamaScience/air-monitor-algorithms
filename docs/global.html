Calculate hourly (diurnal) statistics over recent days.

Returns five arrays of hourly values:

qcType(x, type) → {Array.<(number|null)>}

qcType

Apply a quality-control pass for negative values to an array of measurements. Negative concentrations are not physically meaningful for the data this library handles, so they must be corrected or removed before computing statistics. How aggressively they are removed depends on the QC mode.

Modes:

  • "keep" -- Salvage near-zero readings. Small negatives in [-10, 0) are treated as instrument noise and clamped to 0; values below -10 are considered unrecoverable and set to null (missing).
  • "drop" -- Strict. Any negative value is set to null (missing).

In both modes, non-numeric or missing values (NaN, undefined, null) become null, and non-negative values pass through unchanged.

Parameters:
Name Type Description
x Array.<*>

Input array of values (e.g. PM2.5 concentrations).
type string

QC mode: "keep" or "drop".
Source:

Throws:

Type

Returns:

Type
Array.<(number|null)>
Example
// "keep": clamp small negatives to 0, drop values below -10
qcType([5, -2, -15, null], "keep")  // => [5, 0, null, null]

// "drop": remove all negatives
qcType([5, -2, -15, null], "drop")  // => [5, null, null, null]
